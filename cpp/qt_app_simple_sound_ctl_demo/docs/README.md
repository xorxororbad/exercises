## Description
Project created to exercise some of the technologies involved, not fit for any particular purpose.
A simple application that communicates over dbus with firefox based browsers, can start/stop/play next/prev media content, can be minimized to system tray.
Only some basic commands are implemented, would need more work to be stable and useful.

## Usage
0. Install dependencies. (qt6, cmake, compilers etc).
1. Build the project using build.sh.
2. Run application using run.sh.
